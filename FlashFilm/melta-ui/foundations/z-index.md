# Z-index

> 重なり順序（スタッキング）の統一ルール。5段階に制限し、レイヤーの衝突を防ぐ。

---

## トークン表

| トークン | Tailwind | 値 | 用途 |
|---------|----------|-----|------|
| z-base | `z-0` | 0 | デフォルトコンテンツ |
| z-dropdown | `z-20` | 20 | ドロップダウンメニュー、ツールチップ |
| z-sticky | `z-30` | 30 | 固定サイドバー、sticky ヘッダー |
| z-overlay | `z-40` | 40 | モーダル背景（オーバーレイ） |
| z-modal | `z-50` | 50 | モーダル本体、トースト通知 |

---

## レイヤー構造図

```
z-50  ┌─────────────────────────┐  モーダル本体 / トースト
      │                         │
z-40  ├─────────────────────────┤  オーバーレイ（黒半透明）
      │                         │
z-30  ├─────────────────────────┤  sticky ヘッダー / サイドバー
      │                         │
z-20  ├─────────────────────────┤  ドロップダウン / ツールチップ
      │                         │
z-0   └─────────────────────────┘  通常コンテンツ
```

---

## 用途別ルール

| コンポーネント | トークン | Tailwind | 説明 |
|--------------|---------|----------|------|
| 通常コンテンツ | z-base | `z-0` | デフォルト。明示的に指定する必要はない |
| ドロップダウンメニュー | z-dropdown | `z-20` | select の選択肢一覧 |
| ツールチップ | z-dropdown | `z-20` | ホバー時の補足情報 |
| sticky ナビゲーション | z-sticky | `z-30` | ページ上部の固定ヘッダー |
| 固定サイドバー | z-sticky | `z-30` | レイアウトの固定サイドバー |
| モーダル背景 | z-overlay | `z-40` | `bg-black/50` の半透明レイヤー |
| モーダル本体 | z-modal | `z-50` | ダイアログコンテナ |
| トースト通知 | z-modal | `z-50` | 画面端の通知 |

---

## ルール

- z-index は **5段階のみ** 使用する。`z-10`, `z-[999]` 等は使わない
- モーダルを2層重ねる場合のみ `z-[60]` / `z-[70]` を例外的に許容する（原則禁止）
- `position: relative/absolute/fixed/sticky` を持つ要素にのみ z-index を指定する
- 新しいスタッキングコンテキストの作成は意識的に行う

---

## 禁止パターン

| 禁止 | 理由 | 代替 |
|------|------|------|
| `z-[999]` / `z-[9999]` | マジックナンバー。管理不能になる | 5段階のトークンを使用 |
| `z-10` | 5段階に含まれない | `z-0` または `z-20` |
| 3層以上のモーダル重ね | UXが破綻する | 最大2層まで |
| `z-auto` の多用 | 意図しない重なりが発生する | 明示的なトークン指定 |
