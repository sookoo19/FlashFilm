# FlashFilm 要件定義

## 1. プロダクト概要

- **アプリ名（仮）**：FlashFilm
- **目的**：フラッシュON固定で撮影した写真を自動でフィルム調に補正し、ユーザーに独特なカメラ体験を提供
- **ターゲットユーザー**：カジュアルユーザー、SNSで写真をシェアしたい人

---

## 2. 機能一覧

### P0（必須）

1. **カメラ撮影**
   - フラッシュON固定で撮影
   - 撮影ボタン
   - 撮影後にプレビュー表示
   - 撮影画像を ImageProcessingScreen に渡す

2. **画像加工（MVP）**
   - 固定プリセットによるフィルム調補正
   - 撮影後すぐに加工済みプレビュー表示
   - 端末への保存

---

### P1（ここまで実装することを想定）

1. **AIによる自動補正**
   - サーバー/API経由で画像補正
   - 大元プリセット + ユーザー指定パラメータ（brightness / contrast / grain / warmth）を適用
   - 処理後の画像プレビュー表示
   - 保存 / SNS共有

2. **ユーザー設定（AIQualitySettings）**
   - スライダーで補正の微調整
   - 調整値を端末保存（次回も反映）

---

### P2（可能であれば実装したい）

1. **ギャラリー / 保存画像一覧**
   - 保存済み写真を一覧表示
   - タップで詳細プレビュー
   - 削除 / 共有

2. **追加設定（StorageSettingsなど）**
   - 保存先指定（端末 / クラウド）
   - SNS連携設定
   - テーマやUI設定（ダークモード/ライトモード）

3. **多彩なフィルムプリセット追加**
   - 複数フィルムスタイル（モノクロ、カラー、ヴィンテージなど）
   - AI補正と組み合わせ可能

---

## 3. 技術要件

- **フロント**：React Native + Expo + TypeScript
- **カメラ機能**：`expo-camera` または `react-native-camera`
- **画像加工**：`react-native-image-filter-kit`（固定プリセット）
- **AI補正サーバー**：Python + FastAPI + PyTorch/TensorFlow
- **データ保存**：AsyncStorage / MediaLibrary
- **バージョン管理**：Git + GitHub（機能ブランチ運用）

## 5. 優先度整理

| 優先度 | 内容                                                 |
| ------ | ---------------------------------------------------- |
| P0     | カメラ撮影 + 固定プリセット加工 + 保存               |
| P1     | AI補正 + ユーザー設定（微調整） + SNS共有            |
| P2     | ギャラリー機能 + 保存先設定 + 多彩フィルムプリセット |

---

## 6. MVPゴール

- 「撮る → 自動フィルム補正 → 保存」がスムーズに動作する状態
- AI補正は必須ではないが、可能な限り P1 まで実装
- P2 は完成度向上・UX強化のフェーズ
